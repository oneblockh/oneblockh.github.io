###html基础元素

文档结构标签

•&lt;html&gt;：HTML 文档的根标签，所有其他 HTML 元素都包含在其中。

•&lt;head&gt;：包含文档的元数据，如文档标题、样式表、脚本等，不会在页面中直接显示内容。

•&lt;title&gt;：定义文档的标题，显示在浏览器的标题栏或标签页上。

•&lt;body&gt;：包含文档的可见内容，如文本、图像、链接、表格等，是页面实际展示内容的部分。

文本内容标签

•&lt;h1&gt; - &lt;h6&gt;：标题标签，&lt;h1&gt;是最高级别的标题，&lt;h6&gt;是最低级别，用于设置不同级别的标题文本。


•&lt;p&gt;：段落标签，用于定义段落，将文本分成不同的段落显示。  

•&lt;span&gt;：行内元素，用于在不产生新的布局效果的情况下，对文本进行样式设置或添加脚本行为。  

•&lt;em&gt;：用于强调文本，通常以斜体显示。  •&lt;strong&gt;：表示重要的文本，通常以加粗显示。  

•&lt;br&gt;：换行标签，用于在文本中强制换行。  •&lt;pre&gt;：预格式化文本标签，保留文本中的空格和换行符，按照文本在源代码中的格式显示。  



链接与图像标签  

•&lt;a&gt;：超链接标签，用于创建指向其他页面、文件或位置的链接。  

•&lt;img&gt;：图像标签，用于在页面中插入图像。  

列表标签  

•&lt;ul&gt;：无序列表标签，用于创建无序列表，列表项通常以圆点或其他符号标识。  

•&lt;ol&gt;：有序列表标签，创建有序列表，列表项以数字或其他有序符号标识。  

•&lt;li&gt;：列表项标签，用于定义&lt;ul&gt;或&lt;ol&gt;中的单个列表项。  

表格标签

•&lt;table&gt;：表格标签，用于创建表格。  

•&lt;tr&gt;：表格行标签，定义表格中的一行。  

•&lt;td&gt;：表格数据单元格标签，用于定义表格中的一个单元格，包含表格中的具体数据。  

•&lt;th&gt;：表格标题单元格标签，通常用于表格的表头，默认以粗体显示。  

表单标签  

•&lt;form&gt;：表单标签，用于创建表单，收集用户输入的数据。  

•&lt;input&gt;：输入标签，用于创建各种类型的输入字段，如文本框、密码框、单选按钮、复选框等。  

•&lt;label&gt;：标签标签，用于为表单元素提供描述性文本，与表单元素关联，提高可访问性。  

•&lt;textarea&gt;：文本区域标签，用于创建多行文本输入框。  

•&lt;select&gt;：下拉列表标签，用于创建下拉选择列表。  

•&lt;select&gt;中的可选选项。  其他标签  

•&lt;div&gt;：块级元素，用于将页面划分为不同的区域或块，以便进行布局和样式设置。  

•&lt;iframe&gt;：内联框架标签，用于在当前页面中嵌入另一个 HTML 页面或其他资源。  

•&lt;script&gt;：脚本标签，用于嵌入 JavaScript 代码或引用外部 JavaScript 文件，为页面添加交互性和动态功能。 

•&lt;style&gt;：样式标签，用于定义 HTML 文档的样式表，直接在 HTML 页面中设置样式。
